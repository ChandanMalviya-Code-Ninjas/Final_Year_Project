## 🎨 Design System Updates

### Colors
- **Primary**: Blue (#3b82f6) - Trustworthy healthcare
- **Secondary**: Purple (#8b5cf6) - Innovation
- **Tertiary**: Green (#10b981) - Healthcare/Wellness
- **Neutral**: Slate variations for text/backgrounds

### Gradients
```css
from-blue-500 to-cyan-600
from-purple-500 to-pink-600
from-green-500 to-emerald-600
from-red-500 to-orange-600
from-indigo-500 to-purple-600
```

### Shadows
- **Soft**: 0 4px 12px rgba(0,0,0,0.06)
- **Medium**: 0 10px 25px rgba(0,0,0,0.1)
- **Large**: 0 25px 50px rgba(0,0,0,0.15)
- **Glow**: 0 0 30px rgba(59,130,246,0.6)

### Typography
- **Display Font**: For headings and titles
- **System Font Stack**: For body text
- **Responsive Sizing**: From mobile to 4K

## 🔧 CSS Classes Added

```css
.animate-fade-in-up      /* Fade and slide up */
.animate-fade-in         /* Simple fade */
.animate-slide-in-left   /* Slide from left */
.animate-slide-in-right  /* Slide from right */
.pulse-glow              /* Pulsing glow effect */
.float                   /* Floating animation */
.hover-lift              /* Hover elevation */
.gradient-primary        /* Blue gradient */
.gradient-secondary      /* Purple gradient */
.shadow-soft             /* Subtle shadow */
.shadow-medium           /* Medium shadow */
.shadow-large            /* Large shadow */
.shadow-glow             /* Glow shadow */
.backdrop-blur-*         /* Blur effects */
.text-gradient           /* Gradient text */
```
