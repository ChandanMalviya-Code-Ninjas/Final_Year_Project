## ➕ Add New Medication (3 Steps)

### Step 1: Go to Manage Tab
```
Navigate to: Manage → Add New Medication
```

### Step 2: Fill Form
```
Medication Name:    [Aspirin]
Dosage:            [500mg]
Frequency:         [Daily ▼]
Times:             [08:00] [+Add Time]
Reason:            [Pain relief] (optional)
Notes:             [With water] (optional)
Start Date:        [Today]
```

### Step 3: Save
```
Click: [SAVE MEDICATION]
✓ Confirmation message appears
✓ Medication added to list
```

## 📈 Adherence Formula

```
Adherence = (Doses Taken) / (Doses Scheduled) × 100%

Example:
4 doses taken out of 5 scheduled = 80% adherence
```

### 7-Day Goals
- 📌 Excellent: 95-100%
- ✅ Good:      80-95%
- ⚠️ Fair:      60-80%
- ❌ Poor:      Below 60%
